# BIOS 731 Homework 1: Reproducible Computing

## Overview

This repository contains my submission for Homework 1 in BIOS 731
(Advanced Statistical Computing). The purpose of this project is to
demonstrate a fully reproducible data analysis workflow using R
Projects, Git/GitHub version control, and R Markdown.

The analysis uses a publicly available public health dataset from the
CDC and includes data cleaning, statistical modeling, visualization, and
a reproducible PDF report.

------------------------------------------------------------------------

## Dataset

The dataset used in this project is the **CDC Social Vulnerability Index
(SVI)** dataset.

-   **Source:** Centers for Disease Control and Prevention (CDC)
-   **Description:**\
    The Social Vulnerability Index summarizes socioeconomic and
    demographic factors that influence a community’s ability to prepare
    for, respond to, and recover from public health emergencies. The
    dataset includes measures such as overall social vulnerability,
    population size, and quintile-based rankings.
-   **File location:**\
    `data/CDC_Social_Vulnerability_Index__CDCSVI_.csv`
-   **Access:**\
    The dataset is publicly available and legally shareable for
    educational purposes.

## Repository Structure

```         
.
├── data/                 # Raw input data (read-only)
├── data_clean/           # Cleaned and tidy data generated by scripts
├── R/                    # R scripts for cleaning, modeling, and visualization
├── material/             # Course-related materials (not part of analysis)
├── report.Rmd            # R Markdown source for the final report
├── report.pdf            # Rendered PDF report
├── hw1.Rproj             # R Project file
└── README.md             # Project documentation
```

### R scripts

-   `R/01_load_clean.R`\
    Reads the raw SVI data, performs data cleaning, and saves a tidy
    dataset to `data_clean/`.

-   `R/02_model.R`\
    Fits a linear regression model to assess the association between
    social vulnerability, weighted average quintile, and population
    size.

-   `R/03_visualize.R`\
    Generates a visualization illustrating the relationship between
    weighted average quintile and overall social vulnerability.

------------------------------------------------------------------------

## Reproducibility

To reproduce this analysis from scratch:

1.  Clone this GitHub repository.
2.  Open the project file `hw1.Rproj` in RStudio.
3.  Ensure required R packages are installed (see Session Information
    below).
4.  Render the report:

``` r
rmarkdown::render("report.Rmd")
```

This will regenerate `report.pdf` using only relative file paths and the
provided scripts.
